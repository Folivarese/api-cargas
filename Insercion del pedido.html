Inserción del pedido

Con este método facilitaremos el proceso de facturación a través de Andesmar al ofrecerte la posibilidad de insertar tus pedidos junto con sus números de remito correspondientes.

Request

URL: https://apitest.andesmarcargas.com/

Método: Post

EndPoint: api/InsertPedidoMulti

Body: raw/ Json

Parámetros

Variable Descripción Tipo Obligatorio
CodigoPostalRemitente Código postal del domicilio del remitente Int SI
CalleRemitente Calle del Remitente (solo caracteres alfabéticos) String SI
CalleNroRemitente Número de Calle del Remitente (solo caracteres numéricos) Int SI
NombreApellidoDestinatario Nombre y Apellido del destinatario String SI
CodigoPostalDestinatario Código postal del domicilio del Destinatario Int SI
CalleDestinatario Calle del Destinatario (solo caracteres alfabéticos) String SI
CalleNroDestinatario Calle Nro del Destinatario (solo caracteres numéricos) Int SI
TelefonoDestinatario Teléfono del Destinatario (Hasta 20 caracteres numéricos y carácter +) String SI
Telefono Remitente Teléfono del Remitente (Hasta 20 caracteres numéricos y carácter +) String SI
MailDestinatario Mail del Destinatario String SI
NroRemito Nro. De remito del cliente String SI
Bultos Cantidad de bultos totales del pedido (4 números enteros y 3 decimales) Int SI
Peso Peso total del pedido (4 números enteros y 3 decimales) Double SI
ValorDeclarado Valor Declarado total del pedido Int SI
M3 M3 del pedido (4 números enteros y 8 decimales) Double SI
Alto Alto del pedido (4 números enteros y 3 decimales) Double SI
Ancho Ancho del pedido (4 números enteros y 3 decimales) Double SI
Largo Largo del pedido (4 números enteros y 3 decimales) Double SI
Observaciones Largo del pedido String SI
ModalidadEntregaID Modalidad del pedido (1=Sucursal-Sucursal, 2=Puerta-Puerta, 3=Sucursal-Puerta, 4=Puerta-Sucursal) Int SI
Servicio Si cuenta con el servicio de pago en destino enviar TRUE, caso contrario FALSE Boolean SI
servicio.EsRemitoconformado Si cuenta con el remito conformado enviar TRUE, caso contrario FALSE Boolean SI
Logueo.Usuario Usuario que proporciona Andesmar Cargas String SI
Logueo.Clave Usuario que proporciona Andesmar Cargas String SI
Logueo.CodigoCliente Código del cliente que se va a facturar un pedido String SI
CodigoAgrupacion Código de agrupación que se obtuvo Int SI
