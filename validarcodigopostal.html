Consulta códigos postales

Con este método consultaras la disponibilidad de cobertura de códigos postales habilitados por convenio.

Request

URL: https://apitest.andesmarcargas.com/

Método: Post

EndPoint: api/ConsultaCP

Body: raw/ Json

Parámetros

Variable Descripción Obligatorio
CPOrigen Códogo postal remitente Si
CPDestino Código postal destinatario Si
CodCliente Código del cliente Si

Ejemplo de Request

{ "CPOrigen": "sample string 1", "CPDestino": "sample string 2", "CodCliente": "sample string 3" }

Ejemplo de Response

{ "LocalidaDisponible":"XXXX" "CodigoPostal": "XXXX", "Codigo": "XXXX", "Error": "XXXX" }
