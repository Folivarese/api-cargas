Generacion de etiqueta

Con este método se puede generar una etiqueta propia, la cual se obtiene mediante una URL.

Request

URL: https://apitest.andesmarcargas.com/

Método: Post

EndPoint: api/InsertEtiqueta

Body: raw/ Json

Parámetros

Variable Descripción Obligatorio
CodigoCliente NroRemito Si
CalleRemitente Calle del Remitente (solo caracteres alfabéticos) SI
CalleNroRemitente Nro. de Calle del Remitente (solo caracteres numéricos) SI
CodigoPostalRemitente Código postal del domicilio del remitente SI
NombreApellidoDestinatario Nombre y Apellido del destinatario SI
CodigoPostalDestinatario Código postal del domicilio del Destinatario SI
CalleDestinatario Calle del Destinatario (solo caracteres alfabéticos) SI
CalleNroDestinatario Calle Nro del Destinatario (solo caracteres numéricos) SI
TelefonoDestinatario Teléfono del Destinatario (Hasta 20 caracteres numéricos y carácter +) SI
NroRemito Nro. De remito del cliente SI
Bultos Cantidad de bultos totales del pedido (4 números enteros y 3 decimales) SI
Peso Peso total del pedido (4 números enteros y 3 decimales) SI
ValorDeclarado Valor Declarado total del pedido SI
M3 M3 del pedido (4 números enteros y 8 decimales) SI
Alto Alto del pedido (4 números enteros y 3 decimales) SI
Ancho Ancho del pedido (4 números enteros y 3 decimales) SI
Largo Largo del pedido (4 números enteros y 3 decimales) SI
Observaciones Largo del pedido SI
ModalidadEntrega Modalidad del pedido: Terminal-Terminal, Puerta-Puerta, Terminal-Puerta, Puerta-Terminal SI
UnidadVenta Unidad de Venta habilitada por convenio SI
EsFletePagoDestino Si cuenta con el servicio de pago en destino enviar TRUE, caso contrario FALSE SI
EsRemitoconformado Si cuenta con el remito conformado enviar TRUE, caso contrario FALSE SI
Logueo.Usuario Usuario que proporciona Andesmar Cargas SI
Logueo.Clave Clave que proporciona Andesmar Cargas SI
Logueo.CodigoCliente Código del cliente que se va a facturar un pedido SI
